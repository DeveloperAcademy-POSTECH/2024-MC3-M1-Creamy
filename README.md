# 2024-MC3-M1-Creamy
> 🔮 **Crystal Middle School** (aka. Creamy)

### Commit Convention
``` 
ex) `[Feat/#12] 알림 설정 기능 구현`

 태그      | 설명                                
----------|------------------------------------
 feat     | 새로운 기능 추가                        
 fix      | 버그 수정                             
 refactor | 코드 리팩토링                          
 test     | 테스트                               
 docs     | README와 같은 문서 수정                
 chore    | 관리(maintain), 핵심 내용은 아닌 잡일 등  
 style    | 포맷팅, 세미콜론 누락                   

```

### Branch Naming
``` 
ex) `feat/#24-DataModeling`

태그#이슈번호-기능설명 순으로 작성!

```
